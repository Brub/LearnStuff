# MkDocs

[https://www.mkdocs.org/](https://www.mkdocs.org/)

## Installing MkDocs

```
pip install mkdocs
```

## Getting Started

```
mkdocs new my-project-docs
```

## Launch the built-in dev-server

```
mkdocs serve
```
now you can access the documentation by using the following web address  
```
http://127.0.0.1:8000
```
